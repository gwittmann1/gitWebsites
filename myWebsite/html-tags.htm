HTML tags

<a>
The anchor tag is used for hyperlinks The most useful attribute is href, which is used to add the link.
<base>
The base tag is used to specify the base URL for all URLs in a document
<body>
The body tag is used for documents with text, images, links, etc.
<head>
The div tag is used to create divisions in a document
<div>
The head tag is a contrainer for all the head elements such as titles
<h1>, <h2>, <h3>, ...
These heading tags are used to create headings for titles. Each tag is different than the other one.
<html>
The html tag is used to tell the browser that the document is an HTML document.
<img>
The image tag is used to add images. The most useful attribute is src, which is used to link the image that you want to show
<li>
The line item tag is used to create lists
<link>
The link tag is used to add links.
<meta>
The meta tag is used to provide metadata about the HTML document
<ol>
The ordered list tag is used to create ordered (numerical or alphabetical) lists.
<p>
The paragraph tag is used to create paragraphs.
<script>
The script tag is used to define scripts like JavaScript.
<ul>
The unordered list tag is used to create an unordered list.

Here is a sample HTML document which shows the use of each of these tags:

<!DOCTYPE html>       
<html>
	<head>
		<title></title>
	</head>
	<body>
	</body>
</html>
